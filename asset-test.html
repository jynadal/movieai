Troubleshooting

If any tests fail:

  1. Check if the assets/ folder is uploaded to your server
  2. Verify the folder structure matches your local project
  3. Check file permissions (644 for files, 755 for directories)
  4. Open browser Developer Tools (F12) and check Network tab for 404 errors
  5. Contact your web hosting provider if files exist but still return 404
